PROJECT OBJECTIVE
Create a functional SCADA web application that simulates the electrical and logical operation of a cat food dispenser, from a **120V AC power supply** to the final dispensing mechanism.

The system must be:

- visually realistic
- interactive
- dark theme
- functional in simulation
- coded with **Node.js + npm + React**
- use **lucide-react** for icons
- start on **port 5000**

Use only a modern, simple, and maintainable stack:

- Frontend: React
- Icons: lucide-react
- Backend: Node.js
- Project management: npm
- Web platform: browser application
- The server must listen on **5000**

This could be React + Express with a simple structure.
Ultimately, everything must launch easily with npm. I want a style very close to a modern industrial SCADA/PLC interface:

- very dark background, blue-black/anthracite
- cards/panels with rounded corners
- green/cyan/electric blue light lines
- technical typography in a console/industrial style
- animated states with a subtle glow
- premium, clean, professional appearance
- avoid a "consumer-friendly" style; it needs a true HMI/SCADA look

Desired palette:

- main background: bluish-black
- main text: light gray/off-white
- accents:

- neon green for ON/healthy states

- blue/cyan for titles and power

- yellow/orange for standby/warning

- red for fault/emergency stop/alarm

system name: `CAT_FEEDER_SYS_01`
- version: `v2.4.1`
- node: `N-442`
- simulated uptime
- global system status
- active alarm counter

Example of global system states :
- RUN
- STANDBY
- STOP
- FAULT

2) “ELECTRICAL ONE-LINE” PANEL
Create an animated single-line diagram showing the power supply path:

- `120V AC SUPPLY`
- `MDS-001 MAIN DISCONNECT`
- `CB-001 CIRCUIT BREAKER`
- Power busbar
- `CTR-001 FEEDER CONTACTOR`
- `CTR-002 SOL CONTACTOR`
- `MTR-001 DISPENSER MOTOR`
- `SOL-001 HOPPER GATE`

The diagram must visually show:
- whether power is present
- whether the main disconnect switch is closed/open
- whether the circuit breaker is OK or has tripped
- whether the contactors are energized
- whether the motor is receiving power
- whether the valve/gate is actuated

IMPORTANT :
- The power line must light up green/cyan when the circuit is actually powered.
- If the main disconnect is open, the entire downstream chain must be dead.
- If the circuit breaker is tripped, the downstream power supply must be cut off.
- The electrical behavior must be consistent, not just decorative.

3) “PLC-001 STATUS” PANEL
Create a PLC panel with:
- PLC status
- RUN / STOP / FLT LEDs
- List of digital inputs (DI)
- List of digital outputs (DO)

Digital inputs to simulate:
- `DI-01 MDS-001 ON`
- `DI-02 CB-001 OK`
- `DI-03 HOPPER HIGH`
- `DI-04 HOPPER LOW`
- `DI-05 BOWL DETECT`
- `DI-06 ESTOP NC`

Digital outputs to simulate:
- `DO-01 FEEDER CTR`
- `DO-02 HOPPER SOL`
- `DO-03 BEACON GRN`
- `DO-04 BEACON RED`
Realistic dynamic values:

- Voltage slightly variable around 120V
- Motor current > 0 only when motor is running
- Hopper level decreases with each feed dispensing
- Daily feed counter
